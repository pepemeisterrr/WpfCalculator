# Калькулятор (WPF + MVVM)

Приложение «Калькулятор» выполнено в рамках дисциплины **«Углубленное программирование на C#»** (3 курс).

## Реализованный функционал

- Базовые арифметические операции: `+`, `−`, `×`, `÷`
- Ввод многозначных чисел и дробных чисел с использованием запятой
- Защита от множественного ввода запятой
- Смена знака (`±`)
- Вычисление процента от текущего числа
- Полная очистка (`C`)
- Обработка ошибки деления на ноль
- Сохранение истории вычислений в файл `calculator_history.txt` (в папке Документы)
- Архитектура **MVVM**

## Технологии

- C# (.NET 8)
- WPF (Windows Presentation Foundation)
- Паттерн MVVM
- ООП

## Как запустить

1. Откройте решение в **Visual Studio**
2. Запустите проект (`F5` или `Ctrl + F5`)
